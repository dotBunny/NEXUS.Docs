# Editor Style

<TypeDetails icon="native-class" base="class" type="FNEditorStyle" typeExtra="" headerFile="NexusCoreEditor/Public/NEditorStyle.h" />

Slate style set for the `NexusCoreEditor` module. The class body is generated by the `N_IMPLEMENT_EDITOR_STYLE_HEADER` macro, which expands the standard NEXUS style-set scaffolding (initialisation, shutdown, brush registration, accessor) so every NEXUS plugin's editor module exposes the same style API.

```cpp
class NEXUSCOREEDITOR_API FNEditorStyle
{
    N_IMPLEMENT_EDITOR_STYLE_HEADER("NexusCore", "NEditorStyle")
};
```
