# Console Commands

Some console commands for developers to use provided by `UNWorldAssemblySubsystem`.

| Command | Description | Flags(s) | Shippable|
|:--|:--|:--|:--|
|`N.WorldAssembly.Regenerate`| Tears down the existing worlds `ANCellProxy` and starts a new `UNAssemblyOperation` for all of the `UNOrganComponents` in the world. | `ECVF_Default` | `Yes` |
