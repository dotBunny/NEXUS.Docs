# Actor Pool Set Factory

<TypeDetails icon="ue-object" base="UFactory" type="UNActorPoolSetFactory" typeExtra="" headerFile="NexusActorPoolsEditor/Public/NActorPoolSetFactory.h" />

`UFactory` registered against [UNActorPoolSet](../types/actor-pool-set.md) so the Content Browser's "Create Asset" menu can spawn new pool-set assets. Configured for the standard "create blank asset" path: `bCreateNew = true`, `bEditorImport = false`, `bEditAfterNew = false`. The body wiring (`FactoryCreateNew`, `GetSupportedClass`, ...) is generated by the `N_IMPLEMENT_ASSET_FACTORY` macro from `NexusCoreEditor`'s `NEditorAssetMacros.h`.
