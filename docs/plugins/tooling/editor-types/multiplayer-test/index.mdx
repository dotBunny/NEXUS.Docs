# Multiplayer Test

Editor-only types backing the [Multiplayer Test](../../enhancements/multiplayer-test.md) toolbar button. The [UNMultiplayerTestSubsystem](multiplayer-test-subsystem.md) handles spawning and tracking the configured client/server processes; the [FNMultiplayerTestToolbarSection](multiplayer-test-toolbar-section.md) is the static class that installs the toggle into the Level Editor toolbar. Mirrors the layout of `NexusToolingEditor/Public/MultiplayerTest/`.
