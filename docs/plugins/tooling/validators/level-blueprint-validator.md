# Level Blueprint Validator

A validator that looks at maps to see if any active nodes exist on the `LevelScriptBlueprint`.

:::warning

By default, this validator is **disabled**.

::: 

### Configuration Options

The severity and behaviour of this validator can be configured, project-wide in `Editor Preferences > NEXUS > Tooling > Level: Blueprint Logic`
