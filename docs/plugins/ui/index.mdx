# User Interface

<PluginDetails moduleName="NexusUserInterface" />

This plugin functions as a force multiplier for utilizing UMG:

- **Programatic Widget Updates:** Provide complimentary components which do not trigger their associated event handlers when updated via specific pathways.
- **Default Content:** Often required content to use with your user interface.

## Samples

![NUI](ui-samples.webp)

The `DEMO_NUI` sample map is available once you have enabled the `NEXUS Samples: UI` plugin. This is found in the `NEXUS Samples` category in the `Edit > Plugins` window.

The map demonstrates all of the provided [widgets](types/).
