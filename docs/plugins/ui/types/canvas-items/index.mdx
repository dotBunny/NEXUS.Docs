# Canvas Items

Lightweight native structs that hold the data needed to render a self-contained block on a `FCanvas`. They own measurement, layout, and severity-driven styling so a HUD or overlay can call a single [`FNCanvasUtils`](../canvas-utils.md) entry point per frame. Mirrors the layout of `NexusUI/Public/CanvasItems/`.
