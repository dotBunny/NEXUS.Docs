# Types

Base types shared by every distribution. [PickerParams](picker-params.md) is the projection-aware base struct every shape's `FN<Shape>PickerParams` derives from; the projection defaults it reads from (trace complexity, navmesh query extent, nav agent dimensions) are configured in [Project Settings](../project-settings.md). Per-shape types live under [distributions/](../distributions/).
