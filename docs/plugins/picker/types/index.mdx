# Types

Base types shared by every distribution. [PickerParams](picker-params.md) is the projection-aware base struct every shape's `FN<Shape>PickerParams` derives from; [PickerUtils](picker-utils.md) holds the shared `FCollisionQueryParams` / nav-query defaults and the projection-helper macros the picker implementations expand into. Per-shape types live under [distributions/](../distributions/).
