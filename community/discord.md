We have a brand-new Discord server set up to allow for quicker responses to inquiries and requests for help, so if you have any questions please don’t hesitate to drop by and ask!

https://discord.gg/2M9HczHanW


:::tip[Join the conversation!]

Active discussions happen in `#nexus-general` and `#nexus-support`.

:::

## Server Rules

1. Follow Discord’s own [Community Guidelines](https://discordapp.com/guidelines).
2. Be respectful of each other.
3. Do not spam.
4. Do not impersonate anyone.
5. Listen to and respect the moderators.
6. Do not send unsolicited DMs to any server members.

## Getting Help

When asking for help, please make sure to include what version of the **NEXUS** Framework you are using, right down to the commit hash if possible!
